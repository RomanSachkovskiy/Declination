# Сервис для склонения падежей

Для склонения слова по падежам была использована библиотека https://jitpack.io/#demidko/aot.

По запросу возвращается лист, содержащий падежи слова в следующем порядке:

1. Именительный
2. Родительный
3. Дательный
4. Винительный
5. Творительный
6. Предложный
